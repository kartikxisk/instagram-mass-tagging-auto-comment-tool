📊 Excel File Format

The Excel file should contain usernames to tag in comments. Required columns:

Column Name Description Example
Username Instagram username (without @) john_doe
Is Mentionable TRUE/FALSE - whether to include this user TRUE
Example Excel: 
| Username      | Is Mentionable |
|---------------|----------------|
| user1         | TRUE           |
| user2         | TRUE           |
| spammy_user   | FALSE          |

🚀 How It Works

  1. Select Excel File - Choose your file with usernames to tag
  2. Enter Target Post - Paste the Instagram reel/post URL
  3. Configure Accounts - Add Instagram accounts in the Accounts tab
  4. Add Proxies - (Optional) Add proxies for safety
  5. Start Automation - Click Start and monitor the logs

⚙️ Settings Explained

  • Accounts per Batch: How many accounts to process before a longer break
  • Tags per Account: Maximum tags each account will post
  • Tags per Comment: Number of @mentions in each comment
  • Comments per Account: Number of comments each account will post
  • Pause after X Comments: Take a long break after this many total comments
  • Run Headless: Hide the browser window (recommended for production)

🛡️ Safety Features

  • Random delays between comments (35-120 seconds)
  • Random delays between accounts (5-20 seconds)
  • Long pause after 50 comments (10-20 minutes)
  • Human-like typing and scrolling behavior
  • Automatic detection of blocked/checkpoint accounts
